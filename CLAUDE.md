# Как использовать этот MCP в Claude

## Когда вызывать

Вызывай этот сервер, когда пользователю реально нужно сгенерировать или проверить видео через Seedance.

Подходит:

- text-to-video
- image-to-video
- получить статус генерации
- дождаться готовности видео

Не подходит:

- обсуждение идей без генерации
- простые текстовые ответы

## Как экономить деньги

- не запускай генерацию без явной просьбы
- не дублируй один и тот же prompt без причины
- сначала уточняй/формируй хороший prompt локально, потом уже зови tool
- для polling используй разумный интервал, а не спам

## Как формулировать запрос

Хорошо:

- один prompt
- явное aspect ratio
- явное resolution
- явная duration
- если нужен image-to-video, передай только нужные референсы

Плохо:

- 5 вариантов prompt в одном вызове
- лишние файлы на вход
- бессмысленный polling каждую секунду

## Базовый шаблон

```text
Задача: сделать короткое видео
Prompt: ...
Aspect ratio: 16:9
Resolution: 720p
Duration: 8s
```

## Подсказка по polling

- для видео нормально ждать раз в `5-10` секунд
- если задача не срочная, лучше `8-10` секунд
- если генерация упала, не перезапускай вслепую 10 раз подряд
